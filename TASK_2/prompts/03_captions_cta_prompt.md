# Prompt 03 — Captions & CTAs

## Purpose
Generate platform-optimized captions and CTAs for each ad script — designed to drive clicks, saves, and conversions.

---

## Modular Prompt Template

```
You are a social media copywriter specializing in paid ad captions and CTAs for D2C brands. Write captions that complement UGC-style video ads — not generic brand posts.

Brand: {{brand_name}}
Product: {{product_name}}
Core Benefit: {{product_benefit}}
Offer: {{offer}}
Target Audience: {{target_audience}}
Tone: {{tone}}

Write the following for each platform:

1. Instagram Reels Caption
   - 3–5 lines max (mobile-first)
   - Hook line to stop the scroll in feed
   - 1 benefit statement
   - CTA with offer
   - 5 relevant hashtags

2. YouTube Shorts Caption
   - 2–3 lines
   - Benefit-led opening
   - CTA with link/offer mention

3. Facebook Ad Caption
   - 4–6 lines
   - More descriptive (Facebook audience reads more)
   - Problem → Solution → CTA structure
   - Trust signal included

4. Universal CTA Variations (5 options)
   - Different urgency/angle for A/B testing
   - Max 10 words each

Format clearly with platform labels.
```

---

## Filled Example — GlowLab Vitamin C Serum

```
You are a social media copywriter specializing in paid ad captions for D2C brands. Write captions that feel authentic and conversion-focused — not generic.

Brand: GlowLab Skincare
Product: Vitamin C Brightening Serum
Core Benefit: Visibly brighter skin in 14 days
Offer: Use code GLOW20 for 20% off — link in bio
Target Audience: Women 18–35 with dull or uneven skin tone
Tone: Honest, warm, relatable — like a friend sharing a skincare find

Write captions for: Instagram Reels, YouTube Shorts, Facebook Ads.
Also write 5 universal CTA variations for A/B testing (max 10 words each).
Format with platform labels.
```

---

## Why This Prompt Works

| Element | Reason |
|---|---|
| **Platform-specific format** | Instagram = short, Facebook = longer — different reader behavior |
| **Hook line in caption** | Stops scroll even before video plays (feed view) |
| **5 CTA variations** | Enables A/B testing for best conversion rate |
| **Trust signal instruction** | Adds credibility signal naturally in Facebook copy |
| **Hashtag requirement** | Boosts organic reach on Reels without extra work |

---

## CTA Frameworks — Quick Reference

| Type | Example |
|---|---|
| **Offer-led** | "Use code GLOW20 — 20% off today only" |
| **Urgency** | "Only 200 bottles left — grab yours now" |
| **Soft** | "Check it out — link in bio" |
| **Social Proof** | "Join 50,000+ women who switched to GlowLab" |
| **Question** | "Ready for your glow-up? Link in bio" |
