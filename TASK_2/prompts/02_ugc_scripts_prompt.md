# Prompt 02 — Full UGC Ad Scripts

## Purpose
Generate 3 complete platform-specific UGC ad scripts following the Problem → Agitate → Solution → CTA framework.

---

## Modular Prompt Template

```
You are a UGC ad scriptwriter who writes for real content creators and D2C brands. Write full ad scripts that sound like genuine customer testimonials — not brand copy.

Brand: {{brand_name}}
Product: {{product_name}}
Core Benefit: {{product_benefit}}
Target Audience: {{target_audience}}
Pain Point: {{pain_point}}
Proof / Claim: {{proof_claim}}
Offer / CTA: {{cta}}
Tone: {{tone}}

Write 3 separate UGC ad scripts, one for each platform:

Script 1 — Instagram Reels (30–45 seconds)
Script 2 — YouTube Shorts (45–60 seconds)
Script 3 — Facebook Ads (60 seconds, slightly more detailed)

Each script must follow this structure:
- HOOK: Scroll-stopping first line (2–3 seconds)
- PROBLEM: Relatable pain point (5–8 seconds)
- AGITATE: Make it personal ("I tried everything...")
- SOLUTION: Introduce product naturally, not salesy
- PROOF: Result, timeline, or trust signal
- CTA: Clear, low-friction action

Add speaker notes in [brackets] for tone/delivery guidance.
Format each script clearly with section labels.
```

---

## Filled Example — GlowLab Vitamin C Serum

```
You are a UGC ad scriptwriter who writes for real content creators and D2C brands. Write full ad scripts that sound like genuine customer testimonials — not brand copy.

Brand: GlowLab Skincare
Product: Vitamin C Brightening Serum
Core Benefit: Visibly brighter skin in 14 days
Target Audience: Women 18–35 with dull skin or dark spots
Pain Point: Dull, uneven skin tone despite trying expensive products
Proof / Claim: Dermatologist-tested, 92% users saw brighter skin in 14 days
Offer / CTA: Link in bio — use code GLOW20 for 20% off your first order
Tone: Honest, relatable, conversational — like a friend's recommendation on camera

Write 3 scripts: Instagram Reels (30–45s), YouTube Shorts (45–60s), Facebook Ads (60s).
Each script: HOOK → PROBLEM → AGITATE → SOLUTION → PROOF → CTA.
Add [speaker notes] for delivery. Format with clear section labels.
```

---

## Why This Prompt Works

| Element | Reason |
|---|---|
| **Platform-specific length** | Each platform has different viewer patience and behavior |
| **HOOK→PROBLEM→AGITATE→SOLUTION→PROOF→CTA** | Proven direct response framework used by top agencies |
| **"Not brand copy"** | Forces authentic UGC tone throughout |
| **Speaker notes** | Adds production value — creator knows exactly how to deliver |
| **Proof/claim variable** | Builds credibility without sounding fake |

---

## Platform Script Length Guide

| Platform | Ideal Length | Viewer Behavior |
|---|---|---|
| Instagram Reels | 30–45 seconds | Fast scroll, hook must hit in 2s |
| YouTube Shorts | 45–60 seconds | Slightly more patient, story works well |
| Facebook Ads | 60–90 seconds | Older audience, more detail converts better |
