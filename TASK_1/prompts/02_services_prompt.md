# Prompt 02 — Services Page Copy

## Purpose
Generate service descriptions for each offering of the business — written to inform and convert website visitors.

---

## Modular Prompt Template (Fill Variables Before Use)

```
You are a professional website copywriter specializing in local {{business_type}} businesses. Write persuasive, specific service descriptions — not generic text.

Business Name: {{business_name}}
Location: {{location}}
Tone: {{tone}}

Services to describe:
1. {{service_1_name}} — {{service_1_details}}
2. {{service_2_name}} — {{service_2_details}}
3. {{service_3_name}} — {{service_3_details}}
4. {{service_4_name}} — {{service_4_details}}

For each service, write:
- Service Name (as a heading)
- 2–3 sentence description (benefit-focused, not just feature listing)
- One short tagline (max 8 words)

Format clearly with section labels.
```

---

## Filled Example — Brewed & Beloved Café

```
You are a professional website copywriter specializing in local food & beverage businesses. Write persuasive, specific service descriptions — not generic menu text.

Business Name: Brewed & Beloved Café
Location: Hyderabad, Telangana
Tone: Warm, friendly, slightly poetic — like a food blogger who genuinely loves this place

Services to describe:
1. Specialty Coffee Bar — Single-origin beans, pour-overs, cold brews, classic espresso drinks
2. All-Day Brunch Menu — Eggs, toasts, pancakes, healthy bowls, local-inspired dishes served 8am–4pm
3. Work-Friendly Space — High-speed Wi-Fi, charging points, quiet corners, long-table seating
4. Private Bookings — Small events, birthdays, team offsites up to 30 people

For each service, write:
- Service Name (heading)
- 2–3 sentence benefit-focused description
- One tagline (max 8 words)

Format clearly with section labels.
```

---

## Why This Prompt Works

| Element | Reason |
|---|---|
| **"Not generic text"** | Explicitly blocks bland AI output |
| **Tone reference** | Produces vivid, specific language |
| **Benefit-focused instruction** | Shifts copy from features to customer value |
| **Tagline requirement** | Forces punchy, memorable one-liners per service |
